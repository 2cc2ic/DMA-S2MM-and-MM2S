# DMA-S2MM-and-MM2S-
Build an open source, extremely simple DMA.

我希望构建一个开源可供各位自由配置，极致简单的DMA IP核，来帮助大家更好地完成ZYNQ中大规模地数据传输。

该IP核，它将可以传输任一长度的64/32位数据，通过状态机控制，读写通道突发将使用所能使用的最大突发长度，更有效率的完成读写。
